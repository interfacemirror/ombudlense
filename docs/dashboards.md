# Dashboards & KPI Measurement

All dashboards are built on Dynamics 365 native reporting with Power BI Embedded for advanced analytics.

---

## Executive Dashboard

**Audience:** Council leadership, directors

| KPI | Description | Target |
|---|---|---|
| CSAT | Citizen Satisfaction Score (post-interaction survey) | ≥ 4.2 / 5.0 |
| AHT | Average Handle Time (total interaction duration) | ≤ 6 min |
| AI Call Deflection Rate | % of calls fully handled by AI without agent | ≥ 40% |
| Open Cases | Count of currently open cases | < 500 |
| Closed Cases (Period) | Cases closed in selected period | — |
| SLA Compliance Rate | % of cases resolved within SLA | ≥ 95% |

---

## Citizen Experience Dashboard

**Audience:** Service delivery managers, ombudsman

| KPI | Description | Target |
|---|---|---|
| FCR | First Contact Resolution rate | ≥ 75% |
| Sentiment Trend | Average sentiment score over time | ≥ 0.3 |
| Response Time (Chat) | Median time to first agent/AI response | ≤ 30 sec |
| Response Time (Email) | Median time to first response | ≤ 4 hours |
| Channel Distribution | Breakdown of interactions by channel | — |
| Repeat Contact Rate | % of citizens contacting more than once for same issue | ≤ 15% |

---

## AI Effectiveness Dashboard

**Audience:** AI/technology team, solution architects

| KPI | Description | Target |
|---|---|---|
| Bot Containment Rate | % of sessions fully contained by AI | ≥ 60% |
| Escalation Rate | % of AI sessions escalated to agent | ≤ 40% |
| AI Confidence Score | Average intent confidence at handoff | ≥ 0.85 |
| Intent Recognition Accuracy | % of intents correctly classified | ≥ 92% |
| AI Summary Quality Score | Agent-rated quality of AI briefings | ≥ 4.0 / 5.0 |
